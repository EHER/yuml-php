yuml-php by digitalkaoz

PHP Bindings for http://yuml.me

yuml-php lets you generate an "Class Diagram", "Activity Diagram" or "Use Case Diagram" with the http://yuml.me API. Class Diagram generation works with folder of PHP Classes. You can define several options to the Builder.

Dependencies

kriswallsmith/buzz symfony/finder symfony/console

Install

## Installation with [`composer`](https://github.com/composer/composer.git) ``` json "require"{ "digitalkaoz/yuml-php": ">=0.3" } ```
