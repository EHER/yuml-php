Installation

you have different installations available:

Installation as PHAR Archive

Download the Phar the PHAR here

Use it:

$ php yuml-php.phar

Installation with Composer

Use it in your composer.json

{
    "require" : {
        "digitalkaoz/yuml-php" : ">=0.6"
    }
}

Installation from Source

Clone the Github Repository (git://github.com/digitalkaoz/yuml-php.git)

Install Dependencies with Composer

$ git clone git://github.com/digitalkaoz/yuml-php.git
$ cd yuml-php
$ php composer.phar install
$ bin/yuml-php

Installation as Symfony Bundle

The Symfony2 Bundle is not available yet.

Usage bootstrap-tab.js

Using Class Diagram Builder 


Usage:
    classes [--console] [--debug] [--properties] [--methods] folder basepath

Arguments:
    folder      the folder to scan for classes
    basepath    the basepath for the autoloader

Options:
    --console     log to console
    --debug       debug
    --properties  build with properties
    --methods     build with methods

Using Activity Builder 


Usage:
    activity [--console] [--debug] file

Arguments:
    file          the file to read

Options:
    --console     log to console
    --debug       debug

Using UseCase Builder 


Usage:
    use-case [--console] [--debug] file

Arguments:
    file          the file to read

Options:
    --console     log to console
    --debug       debug
